Sigue estos pasos para registrarte

  1. Proporciona tu información personal
  2. Verificación de identidad
  3. Crea tu usuario y contraseña
  4. Selecciona preguntas de seguridad
  5. Elige el tipo de cuenta
  6. Confirmación del registro

Crear usuario y contraseña

El usuario es requerido. El usuario debe tener al menos 6 caracteres. El usuario no puede tener más de 20 caracteres. El usuario solo puede contener letras, números, puntos, guiones bajos y guiones medios.
La contraseña es requerida. La contraseña debe tener al menos 8 caracteres. La contraseña no puede tener más de 64 caracteres. La contraseña debe contener al menos una letra mayúscula, una letra minúscula, un número y un carácter especial.
La confirmación de la contraseña es requerida. Las contraseñas no coinciden.
